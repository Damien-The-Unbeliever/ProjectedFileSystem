An instance record is

|------0-------|------1-------|------2-------|------3-------|
-------------------------------------------------------------
| Const "FF"   | Flags        |           Reserved          |
-------------------------------------------------------------
|                     Pool Thread Count                     |
-------------------------------------------------------------
|               Concurrent Thread Count                     |
-------------------------------------------------------------
|                                                           |
|                         Instance                          |
|                           Guid                            |
|                                                           |
-------------------------------------------------------------
|                   Name length (UTF 8)                     |
-------------------------------------------------------------
|                   Path length (UTF 8)                     |
-------------------------------------------------------------
| Name bytes followed by path bytes                         |
-------------------------------------------------------------

All integers big-endian
